# Bangalore House Price Prediction

## Overview
Machine Learning project for predicting house prices in Bangalore using historical housing data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask
- HTML/CSS/JavaScript

## Workflow
1. Data Cleaning
2. Feature Engineering
3. Outlier Removal
4. One-Hot Encoding
5. Model Training
6. House Price Prediction

## Model
Linear Regression
